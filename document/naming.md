# 命名ルール

## 大前提

* 読む側が誤解しない名前を付ける
* 省略しない
    * Cd ではなく Code
    * Cnt ではなく Count
* ローマ字での命名はしない
    * 辞書で軽く調べてでも英単語推奨
* 可能であれば型が予想できるように付ける
    * xxxName, xxxNumber など
